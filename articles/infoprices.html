Info Prices


A light weight version of the Prices resources without the ability to trade on quotes. Perfect for any application that just needs to display prices either from single requests or continuous streams.

The Info Price Object

The object contains at least the Quote field group. But it can also hold more information such as display details like x and other stuff like y depending on which field groups you ask for.

Field Groups

Comissions

An object holding all information on comissions related to trading on the price.

DisplayAndformat

An object holding information related to displaying the info prices received like the symbol or formatting.

HistoricalChanges

An object holding information on historical changes based on the current price.

InstrumentPriceDetails

An object holding fields related to the asset type are returned in results. Fields in this group are only returned when issuing a GET and in the initial snapshot if setting up a subscription.

MarketDepth

Market depth fields are returned in results.

PriceInfo

Informational price fields are returned in results.

PriceInfoDetails

Detailed price fields are returned in results.

Quote

Object holding information about prices. Quote data fields are returned in results.
